## create_recipe 函数的触发条件
当前 create_recipe 函数的触发条件是：
'''
POST 请求 → /api/v1/recipes/
请求体： RecipeCreate 数据模型（包含菜品的各个字段）
'''

## 目前它只能通过：
1. Swagger UI (/docs)
2. FastAPI ReDoc (/redoc)
3. curl/Postman
4. 直接的 HTTP POST 请求
来触发。


但是`part-09-async\app\templates\index.html`中的模板已经更改为了在html中可以直接键入符合 RecipeCreate 数据类型的数据了，因此也可以网页端实现存入数据了，
